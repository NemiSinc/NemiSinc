Dashboard for Workplace Fatalities

Workplace Fatalities 

This Dashboard was created with Tableau Desktop. The bubble map shows the number of fatalities and injuries/illnesses for each state. The bubbles are scaled to the number of fatalities, and the color indicates the number of Injuries/Illnesses from light blue to dark blue, from least to most, respectively. The top filled map shows the state ranking for fatalities, and the bottom map shows the ranking for penalties. Color gradient from light blue to dark blue presents lowest state ranking to highest state ranking for each state. The bar graph shows the average amount of penalties in $ for each state.

Dashboard Handgun & Long Gun USA

Handgun & Long Gun USA

This Dashboard was created with Tableau Desktop. The first map compares the average rate for handgun and long gun background checks per 1,000 people in the contiguous United States. The avg. rate for handgun background checks is displayed as a filled map. The avg. rate for long gung background checks is overlaid on the map, displayed as scaled bubbles. The second map looks at the quantity of handgun background checks per county, displayed as scaled bubbles compared to the population size.

Seasonal ARIMA Model 

Seasonality in time series is a regular pattern of changes that repeats over time periods. The seasonal ARIMA Model has the parameters (p,d,1)x(P, D, Q)S. We can find the seasonal parameters by looking at autocorrelation (correlogram and partial correlogram).
